Detailed Description

The NPC-class handles the different kinds of NPCs in the game.

The NPC can be either a student or a professor (More might come). Things that they all have includes name, type(Student, Prof), action(dialog, question, etc), inventory(Keys, test, etc). The best way to use this might be to let two new classes, student and professor, inherit the NPC class.
